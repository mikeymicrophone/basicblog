jScrollPane - scrollTo demo

This page demonstrates how you can control the speed and smoothness of the scrollTo funcitonality of jScrollPane.

You do this by setting different values for animateInterval and animateStep. animateInterval controls how many milliseconds apart animation steps are triggered. animationStep controls the amount of the remaining distance that is traveled on each animation step.

e.g. the default settings of 100 and 3 mean that every 100 milliseconds the scrollpane will jump a third (1/3) of the distance remaining.

This is a very simple type of easing. Probably at some point the plugin should be extended so that it can also use the easing plugin but for now this gives you control over the speed (and smoothness) of your animations...
